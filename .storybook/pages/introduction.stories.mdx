# Introduction

This storybook documents the component library that comes with
[rbb-data/svelte-starter](https://github.com/rbb-data/svelte-starter),
a template project to create data visualizations with [Svelte](https://svelte.dev/)
and [SvelteKit](https://kit.svelte.dev/).

The component library implements designs specified in a rbb|24-specific style guide
[outlined in Figma](https://www.figma.com/file/ADz4iyImo9ikFlsNaOX6MU/Datenteam-Styleguide).
